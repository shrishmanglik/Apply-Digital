# Project Instructions

- Never use any image-generation tool or model. Do not create generated bitmap, WebP, PNG, or SVG-raster assets.
- For hero visuals, posters, backgrounds, and decorative treatments, use pure CSS or existing committed assets only.
- Every update to this prototype must also update GitHub. Commit the intended files and push the active branch to `shrishmanglik/Apply-Digital`.
- Redeploy to Vercel when a change affects the live product experience.
- For documentation-only changes, review `git status` and `git diff` before committing. For product or code changes, run `npm run verify` before pushing unless the blocker is documented.
